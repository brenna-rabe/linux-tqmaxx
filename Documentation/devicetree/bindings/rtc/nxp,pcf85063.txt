* NXP PCF85063		Tiny Real-Time Clock

Required properties:
- compatible: Should contain "nxp,pcf85063".
- reg: I2C address for chip

Optional properties:
- quartz-load-femtofarads : Internal oscillator capacity for quartz crystals
	Values usable for pcf85063 load capacitance are 7000 fF, 12500 fF
- nxp,offset: raw hex value of offset register.
	if not given, driver will keep current register value

Example:
	pcf85063: rtc@51 {
		compatible = "nxp,pcf85063";
		quartz-load-femtofarads = "7000";
		nxp,offset = <0x80>;
		reg = <0x51>;
	};
